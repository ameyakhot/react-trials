# React Web Projects Repository

Welcome to the React Web Projects Repository! This repository contains two distinct web applications built using React.js and various frontend technologies. Below, you'll find a brief overview of each project along with instructions on how to run and deploy them.

## Project 1: React Search Engine

![React Search Engine Screenshot](/screenshots/search-engine.png)

- **Description**: The React Search Engine is a responsive web application that allows users to perform image searches using the Google Search API. It provides a simple and intuitive interface to search for images on the web.

- **Features**:
  - Integration with Google Search API.
  - Responsive design for a seamless user experience on all devices.
  - Customizable search queries.

### Running the React Search Engine

To run the React Search Engine locally, follow these steps:

1. Clone this repository to your local machine.
2. Navigate to the `ReactSearchEngine` directory.
3. Install dependencies using `npm install`.
4. Start the development server with `npm start`.

## Project 2: Finance Landing Page

![Finance Landing Page Screenshot](/screenshots/finance-landing.png)

- **Description**: The Finance Landing Page is a responsive and sleek web page designed for financial services. It utilizes React.js and Tailwind CSS to create an elegant and modern user interface.

- **Features**:
  - Tailwind CSS for rapid and efficient styling.
  - Responsive design for optimal viewing on various screen sizes.
  - Easy-to-customize components for financial services.

### Running the Finance Landing Page

To run the Finance Landing Page locally, follow these steps:

1. Clone this repository to your local machine.
2. Navigate to the `FinanceLandingPage` directory.
3. Install dependencies using `npm install`.
4. Start the development server with `npm start`.
